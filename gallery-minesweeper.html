Minesweeper Game Board: Noah

Description: This group's goal was to design an 8 by 8 button grid that would display images with the user being able to click those images to play the game. This project used a Raspberry Pi 5 and Arduino Nano Every as the brains along with hundreds of pins soldered and thousands of lines of code. In the future they are going to add more games to be played on the board along with the already installed minesweeper
